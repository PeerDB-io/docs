The following list of services are HIPAA Eligible and can be used for storing and processing Protected Health Information (PHI) after signing a Business Associate Agreement (BAA) with PeerDB:
* [PeerDB Cloud](https://app.peerdb.cloud/) with the following restrictions:
    * Mirror Types:
      * CDC
        * Source Peers (with minimum supported PeerDB version in parentheses):
          * PostgreSQL (v0.15.1)
        * Destination Peers (with minimum supported PeerDB version in parentheses):
          * Snowflake (v0.15.1)
          * BigQuery (v0.15.1)
          * Postgres (v0.15.1)
      * QRep
        * Source Peers (with minimum supported PeerDB version in parentheses):
          * PostgreSQL (v0.15.1)
        * Destination Peers (with minimum supported PeerDB version in parentheses):
          * Snowflake (v0.15.1)
          * BigQuery (v0.15.1)
          * Postgres (v0.15.1)
          * ClickHouse (v0.15.1)
          * Elasticsearch (v0.15.1)
